## 🔧 Configuration Steps:

### Step 1: Update frontend/.env.local
Replace the URL with your port 8000 forwarded address:
```env
VITE_API_URL=https://YOUR-CODESPACE-NAME-8000.app.github.dev
```

### Step 2: Update backend/app.py
Add your port 5173 forwarded address to the allow_origins list. Find this section:
```python
app.add_middleware(
    CORSMiddleware,
    allow_origins=[
        "http://localhost:5173",
        "http://localhost:3000",
        "https://YOUR-CODESPACE-NAME-5173.app.github.dev"  # ← ADD THIS LINE
    ],
```

### Step 3: Make port 8000 Public
- Ports tab → right-click **port 8000** → **Port Visibility** → **Public**

### Step 4: Restart Services
- **Backend:** Stop and restart FastAPI
- **Frontend:** Will auto-reload when you save .env.local
